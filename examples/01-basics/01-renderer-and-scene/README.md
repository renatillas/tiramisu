# 01-basics/01-renderer-and-scene

This is the smallest useful Tiramisu application.

It demonstrates:

1. Registering the built-in extensions.
2. Mounting a `tiramisu.renderer`.
3. Nesting a `tiramisu.scene` inside it.
4. Adding one active camera.

Run it with:

```sh
gleam run -m lustre/dev start
```
